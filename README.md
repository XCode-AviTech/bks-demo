# BKS Engineering Services Website

A modern, responsive website for BKS Engineering Services (U) Limited, built to showcase the company’s engineering, security, electrical, and ICT solutions across Uganda.

## Project Overview

This is a static multi-page website with:
- a home page for services, partners, clients, and contact information
- an about page with company background and approach
- a projects page with featured project cards and animated stats

## Project Structure

- index.html — Home page
- about.html — About page
- projects.html — Projects page
- css/ — Stylesheets
  - styles.css — Global styles
  - about.css — About page styles
  - project.css — Projects page styles
  - media.css — Mobile and tablet responsive rules
- js/script.js — Shared JavaScript for navigation, counters, loaders, and carousels
- assets/ — Images, logos, and partner assets


## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
