# BKS Engineering Services (U) Limited - Demo Website

A modern, responsive corporate deno website for **BKS Engineering Services (U) Limited**, showcasing engineering, security, electrical, and ICT solutions across Uganda.

This project is designed as a **client demo website** before full deployment and backend integration.

---

## 🌐 Live Demo: https://xcode-avitech.github.io/bks-demo/


---

## 📌 Project Overview

This website presents the company’s services, projects, and clients in a clean, modern, and professional layout suitable for corporate and government-level presentation.

### Key Sections:
- Hero Section with call-to-action
- About Company Overview
- Services Showcase
- Trusted Clients Slider
- Project Gallery with Lightbox
- Contact Call-to-Action Section
- Sticky Glassmorphism Navbar
- Responsive Footer with Developer Credit

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Variables + Animations)
- Bootstrap 5
- JavaScript (Vanilla)
- Bootstrap Icons
- Lightbox2 (Project Gallery)
